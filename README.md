# libvpx++

A C++ wrapper of [libvpx](http://www.webmproject.org)

## Build and install

### Arch Linux

Download [AUR package](https://aur.archlinux.org/packages/libvpx++).

### Others OS

#### Prerequisite

CMake installed.

#### Command in Linux

##### Build

	cd <path of the source>
	mkdir build
	cd build
	cmake ..
	make

##### Install

	make install
